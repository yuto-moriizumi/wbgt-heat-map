# 開発計画

基本的には、東京暑さ MAP (https://micos-sc.jwa.or.jp/tokyo-wbgt/) の全国版を作る。UI や表示方法が最も優れている。

まずは環境省のデータを利用する形で作成する
電子情報提供サービスの説明： https://www.wbgt.env.go.jp/data_service.php
